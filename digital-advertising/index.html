Digital Advertising Overview

Animation

Flash (R.I.P.)

nike-sport-watch
 Creating animated banner ads used to be much simpler. Flash was a fully featured IDE that compiled small binary files that looked and behaved the same across nearly all browsers. Ads during the Flash era were 40k. Now they are 150k to 250k. Developing ads in HTML is much more complicated and time consuming, but developers are no longer dependent on a single platform.

CSS 3D

galaxy-fold
 All modern browsers have basic support CSS 3D. It's a bit convoluted to use, but has a tiny footprint compared to more robust Javascript 3D libraries. Although not true 3D, a lot can be acheived through careful perspective transformations of 2D elements.

HTML Canvas

galaxy-s10-fortnite
 HTML5 introduced the Canvas element which is meant for direct pixel manipulation using Photoshop-like effects. A canvas can have a 2D or 3D drawing context. Animation can be acheived by drawing new images to the canvas several times per-second. This example uses a Javascript loop to draw particles to the canvas at a rate of 30fps.

Sprite Animation

gs6-butterfly
 Sprite is a generic term for a small bitmap image (usually one of many images stitched together into one large image for better performance). This example uses image sequences stiched together into long strips that give the illusion of motion like a film strip. This setup provides granular control for playing animations forward or in reverse or dynamically changing the frames per-second.

Animated GIF

galaxy-s8-camera
 Sometimes using an animated GIF is still the best way to get the job done. There are many drawbacks to using animated GIFs. They're huge in file size; you can't control or monitor their playback; they only support 8-bit hard transparency and so on. However, if you're really careful, they can still be useful.

After Effects + Lottie

upgrade-to-galaxy
 Lottie is only a few years old but is gaining in popularity. After installing a plugin called Bodymovin in After Effects, animations can be output to a JSON format which can be played back natively on mobile and in a browser using the Lottie JS library. Even though Lottie only supports a subset of After Effects features (works best with vector content, no Expressions or fancy effects), it allows for very high quality animations that compress very well. Lottie animations also allow playback control and dispatch events.

Rich Media / Programmatic

NYTimes Video Masthead

nyt-masthead
 Ads containing video must be run on a rich media platform such as Google's DoubleClick Studio. This ad features a 'draggable' video that spawns smaller videos based on user interaction.

Mercedes - Build Your Own

mbusa-byo
 Dynamic Personalization At Scale: Visitors to the site experience are later retargeted in display channels with dynamic Mercedes-Benz ad content that features an exact, digital replica of the vehicle that they configured on site, including the visitor’s choice of class, model, exterior options and wheels. Using a custom developed solution, new vehicle configurations are pulled dynamically from the MBUSA media server at impression time, and cached at the dynamic ad server for any future requests.

Disney Parks & Resorts

cart-abandoners
 The Shopping Cart Abandoners (SCA) project for Disney Parks & Resorts was a great example of a programmatic campaign done right. The data mostly came from Disney owned properties and was used to help facilitate planning a Disney vacation. The display ad part of the campaign was run through the rich media platform Flashtalking. Flashtalking specializes in ads that use dynamic content (text, images, video, data etc). These ads needed to accomodate many different layouts and scenarios. In this type of setup, one ad template has to do everything (like a Swiss Army Knife).

Mailchimp

wave2
 If a dynamic ad is like a Swiss Army Knife, these ads are like the individual tools. Our client challenged us to build a programmatic campaign without using a rich media platform. Instead of using a dynamic ad setup to create hundreds of versions we created local automations to generate hundreds of ads with static content that could be run as inexpensive standard ads. Rich media platforms such as Flashtalking and DoubleClick Studio help facilitate programmatic audience targeting, but are not required.
